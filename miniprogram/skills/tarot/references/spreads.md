# 牌阵布局参考

## 🃏 单张牌

⏱ **预计解读时长**：~2分钟

```
    ┌─────┐
    │     │
    │  1  │
    │     │
    └─────┘
```
**适用**：每日指引、简单问答、当下能量快照

| 位置 | 含义 |
|------|------|
| 1    | 当前能量/答案/指引 |

## 🃏🃏🃏 三牌阵（3种版本）

⏱ **预计解读时长**：~5分钟

### 版本A：时间轴
```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│  1  │  │  2  │  │  3  │
│过去 │  │现在 │  │未来 │
└─────┘  └─────┘  └─────┘
```
| 位置 | 含义 |
|------|------|
| 1 过去 | 影响当前局面的过去因素 |
| 2 现在 | 当前状态与核心议题 |
| 3 未来 | 若按现有趋势发展的可能走向 |

### 版本B：局面分析
```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│  1  │  │  2  │  │  3  │
│情况 │  │行动 │  │结果 │
└─────┘  └─────┘  └─────┘
```
| 位置 | 含义 |
|------|------|
| 1 情况 | 当前真实处境 |
| 2 行动 | 建议采取的行动方向 |
| 3 结果 | 行动后的可能结果 |

### 版本C：关系解读
```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│  1  │  │  2  │  │  3  │
│ 你  │  │对方 │  │关系 │
└─────┘  └─────┘  └─────┘
```
| 位置 | 含义 |
|------|------|
| 1 你 | 你在此关系中的能量/状态 |
| 2 对方 | 对方的能量/意图/状态 |
| 3 关系 | 两人之间的连接质量与走向 |

## 💎 五牌阵（菱形）

⏱ **预计解读时长**：~7分钟

```
           ┌─────┐
           │  5  │
           │建议 │
           └─────┘
  ┌─────┐  ┌─────┐  ┌─────┐
  │  3  │  │  1  │  │  4  │
  │阻力 │  │核心 │  │潜力 │
  └─────┘  └─────┘  └─────┘
           ┌─────┐
           │  2  │
           │根源 │
           └─────┘
```
| 位置 | 含义 |
|------|------|
| 1 核心（中） | 核心议题或当前能量 |
| 2 根源（下） | 深层原因/潜意识动力 |
| 3 阻力（左） | 挑战/障碍/需要克服的 |
| 4 潜力（右） | 隐藏的资源/优势/机会 |
| 5 建议（上） | 最佳行动方向与结果预示 |

## 🌙 月亮牌阵（4张）

⏱ **预计解读时长**：~6分钟

适合**月度规划**、**能量周期**审视、**新项目的阶段性展望**。

以月相的四个阶段映射一个月（或一个周期）的能量流动：

```
           🌕
        ┌─────┐
        │  3  │
        │满月 │
        └─────┘

🌓                    🌗
┌─────┐          ┌─────┐
│  2  │          │  4  │
│上弦 │          │下弦 │
└─────┘          └─────┘

           🌑
        ┌─────┐
        │  1  │
        │新月 │
        └─────┘
```

| 位置 | 月相 | 含义 |
|------|------|------|
| 1 新月 🌑 | 播种期 | **意图与种子**——这个周期你应该种下什么？新的开始、设定的方向、核心意图 |
| 2 上弦 🌓 | 生长期 | **行动与挑战**——为了让种子生长，你需要付出什么？可能遇到的阻力与需要的行动 |
| 3 满月 🌕 | 高潮期 | **成果与觉察**——这个周期的高潮会带来什么？收获、高光时刻、需要看见的真相 |
| 4 下弦 🌗 | 释放期 | **放下与整合**——什么需要被释放？什么值得被带入下一个周期？ |

**月亮牌阵解读顺序**：
1. 先读牌1（新月），确认这个周期的核心意图
2. 再读牌2（上弦），理解行动与阻力
3. 再读牌3（满月），看见高潮的可能性
4. 最后读牌4（下弦），理解需要放下的
5. 从1→2→3→4串联成一个完整的月度叙事

## ✝️ 凯尔特十字（10张）

⏱ **预计解读时长**：~12分钟

最经典的综合性牌阵，适合复杂问题的多维度解析。

### 布局总览

```
                                ┌─────┐
                                │ 10  │
                                │结果 │
                                └─────┘
                                ┌─────┐
             ┌─────┐            │  9  │
             │  3  │            │希望 │
             │意识 │            │恐惧 │
             └─────┘            └─────┘
             ┌─────┐
  ┌─────┐    │  1  │    ┌─────┐ ┌─────┐
  │  5  │    │ 核心 │    │  6  │ │  8  │
  │近去 │  ──┤  2  ├──  │近未 │ │环境 │
  └─────┘    │ 交叉 │    └─────┘ └─────┘
             └─────┘
             ┌─────┐            ┌─────┐
             │  4  │            │  7  │
             │根基 │            │自我 │
             └─────┘            └─────┘

  ※ 牌2横放在牌1上方，形成十字交叉
```

### 各位置说明

**十字部分（牌1-6）——核心局面**

| 牌位 | 名称 | 放置 | 含义 |
|------|------|------|------|
| 1 | 核心 | 中心 | 问题的核心能量 |
| 2 | 交叉 | 横跨牌1 | 帮助或阻碍的力量（正逆均可） |
| 3 | 意识目标 | 牌1上方 | 你意识层面希望达成的 |
| 4 | 根基过去 | 牌1下方 | 深层过去，奠定当前基础的 |
| 5 | 近期过去 | 牌1左侧 | 正在消退的影响 |
| 6 | 近期未来 | 牌1右侧 | 即将进入的能量 |

**柱部分（牌7-10）——内在到外在、当下到未来**

| 牌位 | 名称 | 放置 | 含义 |
|------|------|------|------|
| 7 | 自我 | 右列最下 | 你的内在状态/出发点 |
| 8 | 环境 | 牌7上方 | 外部因素/他人的影响 |
| 9 | 希望与恐惧 | 牌8上方 | 内心深处的期待与担忧 |
| 10 | 结果 | 最顶部 | 当前趋势下最可能的结局 |

### 解读顺序建议

1. 先读 **牌1+2**（核心张力）——"你的处境"与"横在面前的力量"
2. 再读 **牌3+4**（意识与潜意识）——"你想要的"与"真正的根源"
3. 再读 **牌5+6**（时间轴）——"正在远去的"与"正在到来的"
4. 最后读右列 **牌7→8→9→10**（从内在到外在，从当下到终局）
5. 综合所有牌，讲述一个完整的故事

## 🧭 牌阵推荐指南

| 问题类型 | 推荐牌阵 | 预计时长 |
|----------|----------|----------|
| 每日运势/能量 | 🃏 单张牌 | ~2分钟 |
| 简单是非 | 🃏 单张牌 或 🃏🃏🃏 三牌阵A | ~2-5分钟 |
| 感情问题 | 🃏🃏🃏 三牌阵C 或 ✝️ 凯尔特十字 | ~5-12分钟 |
| 事业决策 | 🃏🃏🃏 三牌阵B 或 💎 五牌阵 | ~5-7分钟 |
| 复杂状况 | ✝️ 凯尔特十字 | ~12分钟 |
| 了解自己/成长 | 💎 五牌阵 或 🐎 马蹄形 | ~7-8分钟 |
| 未来走势 | 🐎 马蹄形 或 🃏🃏🃏 三牌阵A | ~5-8分钟 |
| 月度规划/能量周期 | 🌙 月亮牌阵 | ~6分钟 |
